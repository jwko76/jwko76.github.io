## AI DevOps Framework

자연어(마크다운)로 원하는 기능을 설명하면 Claude AI가 코드를 구현하고,
테스트하고, AWS EC2에 자동 배포하는 프레임워크입니다.

### 주요 기능

- 마크다운 명세 → 자동 코드 구현 (Claude Code CLI)
- pytest 자동 테스트 및 실패 시 롤백
- git 자동 커밋 & 푸시
- AWS EC2 t3.micro 자동 프로비저닝
- 30분 비활성 자동 정지 + Lambda 웨이크업
